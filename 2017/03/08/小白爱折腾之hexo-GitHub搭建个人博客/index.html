小白爱折腾之hexo+GitHub搭建个人博客

参考： http://www.jianshu.com/p/380290deb8f0
http://div.io/topic/1691
http://moxfive.coding.me/yelee/3.Style-Customisation/background.html

前期准备：
1 安装git
2 安装node.js
3 安装hexo
4 申请GitHub账号

搭建步骤：

1 创建仓库：用户名.github.io

2 在本地指定根目录输入命令（cmd或git）：hexo init 用户名.github,io
结果： 生成“用户名.github,io”文件夹
| 用户名.github,io
| |+ node_modules
| |+ scaffolds
| |+ source
| |- themes
| |- landscape
| |- languages
| |- layout
| |- scripts
| |- source
| | .gitgnore
| | _config.yml
| | Rruntfile.js
| | LICENSE
| | package.json
| | README.md
| | .gitgnore
| | _config.yml
| | package.json

3 在“用户名.github,io”根目录输入命令（cmd或git）：hexo s，在浏览器输入localhost:4000

至此，初级搭建成功。

进阶-写博客：

创建新文章： hexo new “文章标题”
